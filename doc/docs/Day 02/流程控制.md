## if 表达式

`if` 表达式允许根据条件执行不同的代码分支。与 JavaScript 等语言不同，`if` 后的条件不会自动转换为布尔值。

```rust
fn main() {
    let number = 3;

    if number {
        println!("number was three");
    }
}
// 这会抛出一个错误
// 4 |     if number {
//   |        ^^^^^^ expected `bool`, found integer
```

### 在 let 中使用 if

`let` 可以使用表达式赋值，而 `if` 正好也是一个表达式，我们可以在 `let` 语句的右侧使用它。

```rust
fn main() {
    let condition = true;

    let number = if condition {
        42
    } else {
        0
    };

    println!("Number is {}", number)
}
```

## 循环

Rust 有三种循环：loop、while 和 for。

### 循环标签

循环标签（loop label），然后将标签与 break 或 continue 一起使用，使这些关键字应用于已标记的循环而不是最内层的循环。

```rust
fn main() {
    let mut count = 0;
    'counting_up: loop {
        println!("count = {}", count);
        let mut remaining = 10;

        loop {
            println!("remaining = {}", remaining);
            if remaining == 9 {
                break;
            }
            if count == 2 {
                break 'counting_up;
            }
            remaining -= 1;
        }

        count += 1;
    }
}
```

### 从循环中返回

循环也是表达式，利用 `break` 可以停止循环并返回出一个值。

```rust
fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;
        if counter == 10 {
            break counter * 2;
        }
    };

    println!("Result is {}", result)
}
```

### 循环遍历

```rust
fn main() {
    let arr = [1,2,3,4,5];

    for item in arr.iter() {
        println!("The value is {}", item)
    }

    for item in (1..4).rev() {
        println!("The tup value is {}", item)
    }
}
```
