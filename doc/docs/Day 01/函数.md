`main` 函数，它是很多程序的入口点。`fn` 关键字，它用来声明新函数。

Rust 代码中的函数和变量名使用 snake case 规范风格。在 snake case 中，所有字母都是小写并使用下划线分隔单词。

```rust
fn main() {
    println!("Hello, world!");
    my_function();
}

fn my_function() {
    println!("Hello, food!")
}
```

## 语句不返回值

语句不返回值。因此，不能把 let 语句赋值给另一个变量，比如下面的例子尝试做的，会产生一个错误：

```rust
fn main() {
    let x = (let y = 6);
}
```

表达式会计算出一个值，函数调用是一个表达式。宏调用是一个表达式。我们用来创建新作用域的大括号（代码块），{}，也是一个表达式，例如：

```rust
fn main() {
    let num = {
        let test = 40;
        test + 2
    };
    println!("Num: {:?}", num);
}
```

> 这里的结尾不加分号表示返回值。

## 具有返回值的函数

函数可以向调用它的代码返回值。我们并不对返回值命名，但要在箭头（->）后声明它的类型。在 Rust 中，函数的返回值等同于函数体最后一个表达式的值。使用 return 关键字和指定值，可从函数中提前返回；但大部分函数隐式的返回最后的表达式。

```rust
fn main() {
    println!("Hello, world!");

    let get_my_num = my_function(42);
    println!("My num: {}", get_my_num);
}

fn my_function(x: i32) -> i32 {
    println!("Hello, food!");
    println!("x is {}", x);

    x
}
```

> 这里的结尾同样不加分号表示返回值。

也可以显式的使用 `return` 关键字返回值。如果结尾不添加分号，则它会从表达式变成语句，也就表示不会返回值。

```rust
fn main() {
    println!("Hello, world!");

    println!("Plus one: {}", plus_one(41))
}

fn plus_one(num: isize) -> isize {
    return num + 1
}
```
