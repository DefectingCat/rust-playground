## 标量类型

标量（scalar）类型代表一个单独的值。Rust 有四种基本的标量类型：整型、浮点型、布尔类型和字符类型。

### 整型溢出

在 debug 模式中，当整形溢出时，Rust 会检查这类问题并使程序 panic。在 release 模式中，Rust 不检测溢出，相反会进行一种被称为二进制补码包装（two’s complement wrapping）的操作。简而言之，值 256 变成 0，值 257 变成 1，依此类推。

## 复合类型

复合类型（Compound types）可以将多个值组合成一个类型。Rust 有两个原生的复合类型：元组（tuple）和数组（array）。

### 元组类型

元组是一个将多个其他类型的值组合进一个复合类型的主要方式。元组长度固定：一旦声明，其长度不会增大或缩小。

```rust
fn main() {
    let tup = (500, 4.2, '😻');
}
```

`tup` 可以使用模式匹配（pattern matching）来解构（destructure）值：

```rust
fn main() {
    let tup = (500, 4.2, '😻');
    let (x, y, z) = tup;
    println!("Z is {}", z);
}
```

除了解构，还可以直接使用 `.` 后跟索引值的语法直接访问他们：

```rust
fn main() {
    let tup = (500, 4.2, '😻');
    println!("Tup one: {}", tup.0);
}
```

### 数组类型

另一个包含多个值的方式是 数组（array）。与元组不同，数组中的每个元素的类型必须相同。

数组的长度是固定的，一旦声明，它们的长度不能增长或缩小。

```rust
fn main() {
    let months = [
        "January",
        "February",
        "March",
        "April",
        "May",
        "June",
        "July",
        "August",
        "September",
        "October",
        "November",
        "December",
    ];
}
```
